# Features Documentation

This directory contains documentation for implemented features in Papr Memory.

## Structure

- **memory_oriented_policies/** - Unified Memory Policy API with node constraints and OMO safety standards
- **schemas/** - Custom schema implementation, schema lifecycle, and schema-related features
- **multi_tenant/** - Multi-tenant architecture and implementation
- **subscriptions/** - Subscription management, rate limits, and tier limits
- **telemetry/** - Telemetry and analytics features
- **temporal/** - Temporal workflow implementations
- **documents/** - Document ingestion, hierarchical chunking, and provider integrations
- **acl/** - Access Control List (ACL) features
- **predictions/** - Predictive memory features
- **holographic_neural_embedding/** - Holographic neural embedding features
- **implementation/** - Implementation summaries and phase completion docs

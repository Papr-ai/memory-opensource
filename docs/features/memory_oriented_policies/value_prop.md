  1. AI-Native Neobank (Ramp but AI-first)                                                                                                                                                                            
                                                                                                                                                                                                                      
  Schema Definition:                                                                                                                                                                                                  
  @schema("ai_neobank")                                                                                                                                                                                               
  class NeoBank:                                                                                                                                                                                                      
      # NODES - Properties only                                                                                                                                                                                       
      @node                                                                                                                                                                                                           
      class Account:                                                                                                                                                                                                  
          id: str = prop(search=exact())                                                                                                                                                                              
          user_id: str = prop(search=exact())                                                                                                                                                                         
          balance: float = prop()                                                                                                                                                                                     
          risk_score: float = prop()                                                                                                                                                                                  
                                                                                                                                                                                                                      
      @node                                                                                                                                                                                                           
      @controlled_vocabulary  # Pre-loaded merchant categories                                                                                                                                                        
      class Category:                                                                                                                                                                                                 
          id: str = prop(search=exact())                                                                                                                                                                              
          name: str = prop(search=semantic(0.90))                                                                                                                                                                     
          risk_level: str = prop()  # low, medium, high, blocked                                                                                                                                                      
                                                                                                                                                                                                                      
      @node                                                                                                                                                                                                           
      class Transaction:                                                                                                                                                                                              
          id: str = prop(search=exact())                                                                                                                                                                              
          amount: float = prop()                                                                                                                                                                                      
          timestamp: datetime = prop()                                                                                                                                                                                
          description: str = prop(search=semantic(0.85))                                                                                                                                                              
          flagged: bool = prop()                                                                                                                                                                                      
                                                                                                                                                                                                                      
      @node                                                                                                                                                                                                           
      class Policy:                                                                                                                                                                                                   
          id: str = prop(search=exact())                                                                                                                                                                              
          name: str = prop(search=semantic(0.85))                                                                                                                                                                     
          threshold: float = prop()                                                                                                                                                                                   
          action: str = prop()                                                                                                                                                                                        
                                                                                                                                                                                                                      
      # EDGES - First-class with full policy                                                                                                                                                                          
      belongs_to = edge(                                                                                                                                                                                              
          Transaction >> Account,                                                                                                                                                                                     
          search=(Account.id.exact(),),                                                                                                                                                                               
          create="auto"                                                                                                                                                                                               
      )                                                                                                                                                                                                               
                                                                                                                                                                                                                      
      in_category = edge(                                                                                                                                                                                             
          Transaction >> Category,                                                                                                                                                                                    
          search=(Category.name.semantic(0.90),),                                                                                                                                                                     
          create="never",  # Only link to existing categories                                                                                                                                                         
          when=Category.risk_level != "blocked"                                                                                                                                                                       
      )                                                                                                                                                                                                               
                                                                                                                                                                                                                      
      triggers_policy = edge(                                                                                                                                                                                         
          Transaction >> Policy,                                                                                                                                                                                      
          search=(Policy.name.semantic(0.85),),                                                                                                                                                                       
          create="never",                                                                                                                                                                                             
          when=And(                                                                                                                                                                                                   
              Transaction.amount > Policy.threshold,                                                                                                                                                                  
              Not(Transaction.flagged == True)                                                                                                                                                                        
          )                                                                                                                                                                                                           
      )                                                                                                                                                                                                               
                                                                                                                                                                                                                      
  What This Enables:                                                                                                                                                                                                  
                                                                                                                                                                                                                      
  # Every transaction automatically:                                                                                                                                                                                  
  # 1. Links to account (graph structure)                                                                                                                                                                             
  # 2. Gets categorized against controlled vocabulary (no hallucinated categories)                                                                                                                                    
  # 3. Checks against policies via graph traversal                                                                                                                                                                    
  # 4. Conditional flagging based on when/set                                                                                                                                                                         
                                                                                                                                                                                                                      
  await client.add_memory(                                                                                                                                                                                            
      content="$847.32 purchase at CryptoExchange.io for user acct_123",                                                                                                                                              
      link={                                                                                                                                                                                                          
          Transaction.description.semantic(0.85)                                                                                                                                                                      
              .in_category()  # Uses schema default - finds "Cryptocurrency" category                                                                                                                                 
              .triggers_policy()  # Traverses to matching policies                                                                                                                                                    
              .when(Category.risk_level == "high")                                                                                                                                                                    
              .set({                                                                                                                                                                                                  
                  Transaction.flagged: True,                                                                                                                                                                          
                  Transaction.review_reason: Auto()                                                                                                                                                                   
              })                                                                                                                                                                                                      
      }                                                                                                                                                                                                               
  )                                                                                                                                                                                                                   
                                                                                                                                                                                                                      
  # Query: "Find all transactions at high-risk merchants for this user"                                                                                                                                               
  # Graph traversal: Account -[BELONGS_TO]- Transaction -[IN_CATEGORY]-> Category(risk="high")                                                                                                                        
                                                                                                                                                                                                                      
  Value Created:                                                                                                                                                                                                      
  - Without Papr: 6-12 months to build transaction categorization, policy engine, fraud detection rules, audit trail                                                                                                  
  - With Papr: Schema defines it all declaratively. Add memories, graph handles the rest.

  2. AI Marketing/Sales (Rev.ai on steroids)                                                                                                                                                                          
                                                                                                                                                                                                                      
  Schema Definition:                                                                                                                                                                                                  
  @schema("ai_sales_platform")                                                                                                                                                                                        
  class AISales:                                                                                                                                                                                                      
      @node                                                                                                                                                                                                           
      class Company:                                                                                                                                                                                                  
          id: str = prop(search=exact())                                                                                                                                                                              
          name: str = prop(search=semantic(0.90))                                                                                                                                                                     
          size: str = prop()  # smb, mid, enterprise                                                                                                                                                                  
          industry: str = prop()                                                                                                                                                                                      
          icp_score: float = prop()                                                                                                                                                                                   
                                                                                                                                                                                                                      
      @node                                                                                                                                                                                                           
      class Contact:                                                                                                                                                                                                  
          email: str = prop(search=exact())                                                                                                                                                                           
          name: str = prop(search=semantic(0.90))                                                                                                                                                                     
          title: str = prop(search=semantic(0.85))                                                                                                                                                                    
          engagement_score: float = prop()                                                                                                                                                                            
                                                                                                                                                                                                                      
      @node                                                                                                                                                                                                           
      @controlled_vocabulary                                                                                                                                                                                          
      class Intent:                                                                                                                                                                                                   
          id: str = prop(search=exact())                                                                                                                                                                              
          name: str = prop(search=semantic(0.90))  # "pricing_inquiry", "competitor_mention", "budget_discussion"                                                                                                     
          strength: str = prop()  # low, medium, high, buying                                                                                                                                                         
                                                                                                                                                                                                                      
      @node                                                                                                                                                                                                           
      @controlled_vocabulary                                                                                                                                                                                          
      class Stage:                                                                                                                                                                                                    
          id: str = prop(search=exact())                                                                                                                                                                              
          name: str = prop(search=semantic(0.85))                                                                                                                                                                     
          order: int = prop()                                                                                                                                                                                         
                                                                                                                                                                                                                      
      @node                                                                                                                                                                                                           
      class Interaction:                                                                                                                                                                                              
          id: str = prop(search=exact())                                                                                                                                                                              
          content: str = prop(search=semantic(0.85))                                                                                                                                                                  
          channel: str = prop()  # email, call, meeting, website                                                                                                                                                      
          timestamp: datetime = prop()                                                                                                                                                                                
                                                                                                                                                                                                                      
      # EDGES with full policy                                                                                                                                                                                        
      works_at = edge(                                                                                                                                                                                                
          Contact >> Company,                                                                                                                                                                                         
          search=(Company.name.semantic(0.90),),                                                                                                                                                                      
          create="auto"                                                                                                                                                                                               
      )                                                                                                                                                                                                               
                                                                                                                                                                                                                      
      shows_intent = edge(                                                                                                                                                                                            
          Interaction >> Intent,                                                                                                                                                                                      
          search=(Intent.name.semantic(0.90),),                                                                                                                                                                       
          create="never",  # Only controlled intent signals                                                                                                                                                           
          when=Intent.strength.exists()                                                                                                                                                                               
      )                                                                                                                                                                                                               
                                                                                                                                                                                                                      
      at_stage = edge(                                                                                                                                                                                                
          Contact >> Stage,                                                                                                                                                                                           
          search=(Stage.name.semantic(0.85),),                                                                                                                                                                        
          create="never"                                                                                                                                                                                              
      )                                                                                                                                                                                                               
                                                                                                                                                                                                                      
      # Cross-entity intelligence via graph traversal                                                                                                                                                                 
      has_buying_signal = edge(                                                                                                                                                                                       
          Company >> Intent,                                                                                                                                                                                          
          search=(Intent.name.semantic(0.90),),                                                                                                                                                                       
          create="never",                                                                                                                                                                                             
          when=Intent.strength == "buying"                                                                                                                                                                            
      )                                                                                                                                                                                                               
                                                                                                                                                                                                                      
  What This Enables:                                                                                                                                                                                                  
                                                                                                                                                                                                                      
  # Every interaction automatically:                                                                                                                                                                                  
  # 1. Links to contact/company                                                                                                                                                                                       
  # 2. Detects intent signals from controlled vocabulary                                                                                                                                                              
  # 3. Updates stage based on conditions                                                                                                                                                                              
  # 4. Propagates signals to company level                                                                                                                                                                            
                                                                                                                                                                                                                      
  await client.add_memory(                                                                                                                                                                                            
      content="""                                                                                                                                                                                                     
      Call with Sarah Chen (VP Engineering at Acme Corp):                                                                                                                                                             
      - Discussed our pricing tiers in detail                                                                                                                                                                         
      - Mentioned they're evaluating vs Competitor X                                                                                                                                                                  
      - Asked about enterprise security features                                                                                                                                                                      
      - Timeline: Q2 budget approval                                                                                                                                                                                  
      """,                                                                                                                                                                                                            
      link={                                                                                                                                                                                                          
          # Interaction extracts intent signals                                                                                                                                                                       
          Interaction.content.semantic(0.85)                                                                                                                                                                          
              .shows_intent(Intent.name.semantic(0.95))                                                                                                                                                               
              .set({Interaction.channel: "call"}),                                                                                                                                                                    
                                                                                                                                                                                                                      
          # Contact gets engagement boost                                                                                                                                                                             
          Contact.name.semantic(0.90)                                                                                                                                                                                 
              .works_at(Company.name.semantic(0.90))                                                                                                                                                                  
              .when(Intent.strength == "high")                                                                                                                                                                        
              .set({                                                                                                                                                                                                  
                  Contact.engagement_score: Auto(),                                                                                                                                                                   
                  Contact.last_high_intent: "2026-01-25"                                                                                                                                                              
              }),                                                                                                                                                                                                     
                                                                                                                                                                                                                      
          # Company-level buying signal via graph traversal                                                                                                                                                           
          Company.name.semantic(0.90)                                                                                                                                                                                 
              .has_buying_signal()                                                                                                                                                                                    
              .when(                                                                                                                                                                                                  
                  And(                                                                                                                                                                                                
                      Intent.name.contains("pricing"),                                                                                                                                                                
                      Company.size == "enterprise"                                                                                                                                                                    
                  )                                                                                                                                                                                                   
              )                                                                                                                                                                                                       
              .set({Company.icp_score: Auto()})                                                                                                                                                                       
      }                                                                                                                                                                                                               
  )                                                                                                                                                                                                                   
                                                                                                                                                                                                                      
  # Query: "Find all enterprise companies where ANY contact showed buying intent"                                                                                                                                     
  # Graph: Company(size="enterprise") <-[WORKS_AT]- Contact -[SHOWS_INTENT]-> Intent(strength="buying")                                                                                                               
                                                                                                                                                                                                                      
  Value Created:                                                                                                                                                                                                      
  - Traditional approach: Build lead scoring model, intent detection ML, CRM integration, account rollup logic                                                                                                        
  - With Papr: Intent vocabulary controlled, graph traversal handles account-level intelligence, policies automate scoring

  From https://www.bvp.com/atlas/roadmap-ai-systems-of-action:                                                                                                                                                        
                                                                                                                                                                                                                      
  "AI agents are offloading much of the manual labor that surrounds a System of Record... We are seeing this pattern recur in many markets where the SoR is closely tied to high-volume human communications or

  Papr's opportunity: Don't be a memory layer for existing SoRs.   The schema + edges + policies design means:                                                                                                                                                                         
  1. The graph structure IS the data model (like a schema in traditional DB)

  3. The memories ARE the records (source of truth, not cache)                                                                                                                                                        
  4. The agents ARE the UI (query and update via natural language)

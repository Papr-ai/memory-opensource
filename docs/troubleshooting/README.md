# Troubleshooting

This directory contains troubleshooting guides, fix summaries, and solutions to common issues.

## Contents

- **Fix Summaries** - Documentation of bugs fixed and their solutions
- **Troubleshooting Guides** - Step-by-step guides for resolving issues
- **Workarounds** - Temporary solutions while permanent fixes are being developed

## Common Issues

- **Parse Server**: File upload issues, connection problems
- **MongoDB**: SSL configuration, connection latency
- **Memory Storage**: Chunk IDs, relevance scores, versioning
- **Workers**: Worker startup and execution issues
